Description

This project hosts the web front-end for Seven Labs Messaging Protocol. It was created in PHP 5.3 and remains compliant with HTML5 and CSS3. Currently the project remains inactive while we work on the back end of SLMP.

Prerequisites

  • PHP 5.3+
  • PHP Modules: cURL, mysql/sqlite
  • MySQL 5.5+ (or some other data source compatible with Syslog-DB formatting)
